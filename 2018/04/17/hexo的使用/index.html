hexo的使用

发表于 2018-04-17 | 分类于 Hexo

hexo使用基本命令

  1. 在windows终端中 D: blog下运行 输入hexo s，开启服务器，访问该网址，正式体验Hexo，使用当地网址，localhost:4000就能看效果了。
  2. 问题：假如页面一直无法跳转，那么可能端口被占用了。此时我们ctrl+c停止服务器，接着输入“hexo server -p 端口号”来改变端口号
  3. 新建一篇博客，在cmd执行命令：hexo new post “博客名”
  4. 这时候在文件夹_posts目录下将会看到已经创建的文件
  5. 使用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了， 推从到github上了。
  6. 部署成功后访问你的地址：http://用户名.github.io。那么将看到生成的文章。
  7. 博客文章的编写，①在cmd终端D: blog下执行文件：hexo new post “文章名”生成在blog/source/_posts目录下。②打开这个文件并编写内容并保存③在终端hexo s在本地浏览器参看效果④没问题之后同步到github上输入命令：hexo d -g。
  8. 对右侧栏里的菜单项进行跳转或页面的生成显示，菜单项是在next主题文件中修改的menu字段，对这些菜单栏中的字段点击后进行跳转的页面展示，在cmd终端D: blog下输入命令: hexo new page categories =====>>在blog/source下就新生成一个caregories目录，目录中的index.md文件就是跳转之后显示的文件代码，对这个文件进行加工，显示效果。
  9. 文章在首页下拉显示的控制： 在编写的md文档中使用 <!– more –> 手动进行截断，想显示多少就显示多少。
